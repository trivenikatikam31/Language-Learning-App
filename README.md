# Language Learning App

## Overview

The Language Learning App helps users learn new languages through vocabulary flashcards, phrases, grammar lessons, and quizzes.

## Features

* Language selection
* Vocabulary flashcards
* Translation support
* Pronunciation guides
* Quiz and score tracking
* Progress saving with localStorage
* Responsive and modern UI

## Technologies Used

* HTML
* CSS
* JavaScript

## Learning Outcomes

* DOM Manipulation
* Local Storage
* Interactive UI Design
* JavaScript Event Handling
